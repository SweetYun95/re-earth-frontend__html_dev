시나리오 주제 : 핀테크
내용 : 탄소절감을 이용한 포인트 적립과 포인트샵에서 아이템 구매와 기부까지
컬러:
메인 컬러 RGB(146,195,164), RGB(242,246,235)
서브 컬러 RGB(86,130,101), RGB(219,245,142)
포인트 컬러 RGB(255,206,85), RGB(251,110,82)
텍스트 컬러 RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

상품 리스트 페이지 디자인
1. 좌측 사이드 메뉴
   - 상단: 유저 마이페이지 아이콘(프로필 원형 아바타)
   - 메뉴 버튼: "포인트 샵", "아나바다 장터"
   - 서브 메뉴: "상품등록", "상품수정" 버튼
   - 배경: 포인트 컬러 RGB(255,206,85), 선택된 메뉴는 서브 컬러 RGB(86,130,101) 강조

2. 본문 영역
   - 상단: 상품 대표 이미지 및 카드 요약 정보
      · 이미지 영역(사각 프레임)
      · 업체명 / 상품명 / 금액 표시
   - 하단: 상품 리스트 카드 반복 배치 (그리드 3~4열)
      · 상품 이미지
      · 상품명, 가격
      · 하트/찜 아이콘 (선택 시 포인트 컬러 RGB(251,110,82))
   - 카드 디자인: 흰색 배경, 그림자 효과, 모서리 라운딩

3. 포인트샵 / 아나바다 장터 화면 구분
   - 포인트샵: “포인트샵 화보/아이템 이미지” 강조
   - 아나바다 장터: 이미지 + 제목/금액/판매자 이름을 카드에 배치

4. 배경 및 장식
   - 전체 배경: 메인 컬러 RGB(242,246,235)
   - 주요 타이틀: 메인 컬러 RGB(146,195,164)
   - 보조 안내 텍스트: 텍스트 컬러 RGB(158,158,158)

요청 스타일:
- 핀테크 플랫폼다운 신뢰성
- 친환경 & 기부 분위기 강조
- 직관적인 상품 탐색과 쇼핑 경험
