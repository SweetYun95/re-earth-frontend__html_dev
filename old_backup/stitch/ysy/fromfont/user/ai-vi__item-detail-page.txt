시나리오 주제 : 핀테크
내용 : 탄소절감을 이용한 포인트 적립과 포인트샵에서 아이템 구매와 기부까지
컬러:
메인 컬러 RGB(146,195,164), RGB(242,246,235)
서브 컬러 RGB(86,130,101), RGB(219,245,142)
포인트 컬러 RGB(255,206,85), RGB(251,110,82)
텍스트 컬러 RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

상품 상세 페이지 디자인
1. 좌측 사이드 메뉴
   - 상단: 유저 마이페이지 아이콘
   - 메뉴 버튼: "포인트 샵", "아나바다 장터"
   - 배경: 포인트 컬러 RGB(255,206,85)
   - 선택된 메뉴는 서브 컬러 RGB(86,130,101) 강조
   - 메뉴 텍스트: 기본 텍스트 컬러 RGB(51,51,51)

2. 상품 이미지 영역
   - 중앙 정렬된 이미지 박스
   - 좌우 슬라이더 버튼(〈, 〉) 배치
   - 이미지 박스 테두리는 라운딩 처리
   - 배경은 흰색, 그림자 효과 적용

3. 상품 정보 영역
   - 상품명: 큰 글씨, 텍스트 컬러 RGB(51,51,51)
   - 구분선(얇은 라인)으로 구획
   - 가격: “가격: 1000p” 같은 형태, 포인트 컬러 RGB(251,110,82)로 강조

4. 수량 조절 및 구매
   - 수량 선택: “– n개 +” 버튼 형태, 버튼은 서브 컬러 RGB(219,245,142)
   - 구매 버튼: “상품 구매” 버튼, 포인트 컬러 RGB(255,206,85) 배경 + 흰색 텍스트
   - 버튼은 라운딩 처리, 살짝 입체감(그림자) 부여

5. 배경 및 전체 스타일
   - 페이지 전체 배경: 메인 컬러 RGB(242,246,235)
   - 주요 타이틀 배경: 메인 컬러 RGB(146,195,164)
   - 안내문/보조 텍스트: RGB(158,158,158)

요청 스타일:
- 깔끔하고 현대적인 UI
- 친환경적인 무드와 신뢰성 강조
- 상품 정보가 직관적으로 보이는 구조
