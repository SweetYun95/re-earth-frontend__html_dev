시나리오 주제: 핀테크 고객센터
내용: 유저가 직접 문의글을 작성하고 내역을 확인할 수 있는 1:1 문의 게시판
컬러:
- 메인 컬러: RGB(146,195,164), RGB(242,246,235)
- 서브 컬러: RGB(86,130,101), RGB(219,245,142)
- 포인트 컬러: RGB(255,206,85), RGB(251,110,82)
- 텍스트 컬러: RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

유저 1:1 문의 게시판 디자인:
레이아웃:
- 좌측 사이드바: 고객센터 메뉴(FAQ, 1:1문의, 공지사항)
- 우측 메인 영역: 문의 작성 + 문의 내역 리스트

화면 구성:
1. 문의 작성 폼
   - 문의 유형 (select)
   - 제목 (input)
   - 내용 (textarea)
   - 파일 첨부 (옵션)
   - [작성하기] 버튼
2. 문의 내역 리스트
   - 표 형식: 제목, 작성일, 처리상태
   - 상태 색상:
     · 대기: 서브 컬러 RGB(219,245,142)
     · 완료: 메인 컬러 RGB(146,195,164)
3. 상세 조회
   - 제목, 작성자, 작성일
   - 문의 본문
   - 관리자 답변 영역
4. 디자인 스타일
   - 카드형 배경, 흰색 + 그림자
   - 버튼 강조: 포인트 컬러 RGB(251,110,82)
