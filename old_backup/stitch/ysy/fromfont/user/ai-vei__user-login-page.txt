시나리오 주제 : 핀테크
내용 : 탄소절감을 이용한 포인트 적립과 포인트샵에서 아이템 구매와 기부까지
컬러:
메인 컬러 RGB(146,195,164), RGB(242,246,235)
서브 컬러 RGB(86,130,101), RGB(219,245,142)
포인트 컬러 RGB(255,206,85), RGB(251,110,82)
텍스트 컬러 RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

로그인 페이지 디자인
1. 상단 영역(Header)
   - 로고와 서비스 이름 배치
   - 메인 컬러 RGB(146,195,164) 배경
   - 브랜드 신뢰감을 줄 수 있는 간결한 톤

2. 본문(로그인 폼)
   - 입력 필드: 이메일, 비밀번호
   - 체크박스: "자동 로그인" 옵션
   - 텍스트 컬러 기본 RGB(51,51,51), 보조 안내 텍스트 RGB(158,158,158)
   - 비밀번호 찾기 / 회원가입 링크: RGB(158,158,158)

3. CTA(Call to Action) 버튼
   - “로그인” 버튼: 포인트 컬러 RGB(251,110,82), 텍스트는 흰색
   - 소셜 로그인 버튼(Google, Kakao): 서브 컬러 RGB(86,130,101) 배경, 흰색 텍스트

4. 배경 및 장식
   - 전체 배경: RGB(242,246,235)
   - 친환경 아이콘(나무, 지구, 잎사귀 등)을 서브 컬러 RGB(219,245,142) 포인트로 배치
   - 로그인 박스는 카드 형태(흰색 배경, 살짝 둥근 모서리, 그림자 효과)

요청 스타일:
- 현대적이고 신뢰성 있는 핀테크 서비스 이미지
- 간결하면서도 따뜻한 친환경 무드
- 로그인 과정의 직관성과 접근성을 강조
