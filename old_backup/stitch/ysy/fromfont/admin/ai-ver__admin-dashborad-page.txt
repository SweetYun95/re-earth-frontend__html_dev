시나리오 주제: 핀테크
내용: 탄소절감을 이용한 포인트 적립과 포인트샵에서 아이템 구매와 기부까지

컬러 팔레트:
- 메인 컬러: RGB(146,195,164), RGB(242,246,235)
- 서브 컬러: RGB(86,130,101), RGB(219,245,142)
- 포인트 컬러: RGB(255,206,85), RGB(251,110,82)
- 텍스트 컬러: RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

관리자 메인 대시보드 화면 디자인:
1. 좌측 사이드 메뉴
   1.1 상단: 관리자 프로필 아이콘
   1.2 메뉴 항목:
       - 대시보드 설정 (→ 위젯 구성 커스텀 화면으로 이동)
       - 유저 관리 페이지
       - 대중교통 연동 관리
       - 커뮤니티 관리
       - 통계
       - 기부 관리
       - 포인트샵 관리
       - 제휴사 관리
       - 설정
   1.3 현재 활성화 메뉴: "대시보드 메인"
   1.4 배경: 포인트 컬러 RGB(255,206,85)
   1.5 활성화 메뉴 강조: 서브 컬러 RGB(86,130,101)

2. 본문 영역 (메인 보드판)
   2.1 화면 전체가 위젯 보드 레이아웃으로 구성
   2.2 위젯 예시:
       - 통계 그래프 1 (일간 포인트 적립량)
       - 통계 그래프 2 (유저별 기여도 분포)
       - 기부 순위 (상위 기부자/단체 리스트)
       - 실시간 접속자 수
       - 최근 게시물 (커뮤니티 최신 글)
       - 문의글 현황 (처리 전/후 상태 구분)
   2.3 위젯은 카드 형태, 자유롭게 재배치 가능하다는 느낌 강조

3. 위젯 카드 디자인
   3.1 배경: 흰색
   3.2 효과: 그림자, 둥근 모서리
   3.3 제목: 상단 고정 (예: "통계 그래프1")
   3.4 내부: 그래프, 표, 리스트, 숫자 지표 표시
   3.5 중요 수치 강조 색: 포인트 컬러 RGB(251,110,82)

4. 배경 및 스타일
   4.1 전체 배경: 메인 서브 컬러 RGB(242,246,235)
   4.2 위젯 간 여백 충분히 확보, 격자형 정렬
   4.3 텍스트:
       - 기본: RGB(51,51,51)
       - 보조 안내: RGB(158,158,158)
   4.4 그래프/차트 색상: 메인 & 서브 컬러 조합

5. 요청 스타일
   5.1 관리자 친화적인 대시보드 스타일
   5.2 각 섹션을 위젯처럼 모듈화 → 커스터마이즈 가능성 시각적으로 암시
   5.3 데이터 가독성 강조
   5.4 친환경 핀테크 서비스 아이덴티티 반영
