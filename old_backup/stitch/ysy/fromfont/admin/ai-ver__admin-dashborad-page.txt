 시나리오 주제 : 핀테크 
내용 : 탄소절감을 이용한 포인트 적립과 포인트샵에서 아이템 구매와 기부까지
컬러:
메인 컬러 RGB(146,195,164), RGB(242,246,235)
서브 컬러 RGB(86,130,101), RGB(219,245,142)
포인트 컬러 RGB(255,206,85), RGB(251,110,82)
텍스트 컬러 RGB(51,51,51), RGB(158,158,158), RGB(245,245,245), RGB(255,255,255)

유저 마이페이지 – 프로필 수정 화면 디자인
1. 좌측 사이드 메뉴
   - 현재 화면과 동일한 구조 유지
   - 상단: 유저 프로필 아이콘, 닉네임, 랭크
   - 메뉴 항목: "Public Transport", "Old Clothes Donation", "Point Shop", "Edit Profile"
   - 현재 활성화: "Edit Profile"
   - 배경: 포인트 컬러 RGB(255,206,85)
   - 활성화 메뉴 강조: 서브 컬러 RGB(86,130,101)

2. 본문 영역
   - 상단 타이틀: “프로필 수정”
   - 프로필 이미지 변경 섹션
     · 현재 이미지 미리보기
     · “이미지 변경” 버튼 (포인트 컬러 RGB(251,110,82), 흰색 텍스트)
   - 입력 필드
     · 닉네임 입력창
     · 이메일 (수정 불가, 비활성화 상태)
     · 비밀번호 변경: 새 비밀번호 / 비밀번호 확인
     · 자기소개 (짧은 텍스트 에어리어)
   - CTA 버튼
     · “저장하기” 버튼: 메인 컬러 RGB(146,195,164), 흰색 텍스트
     · “취소” 버튼: 텍스트 컬러 RGB(158,158,158)

3. 보조 정보 영역 (우측 혹은 하단 카드)
   - 내 포인트 현황: “누적 포인트: N P”
   - 환경 기여도: “이번 달 탄소 절감: X kg CO₂”
   - 흰색 카드, 그림자 효과, 둥근 모서리

4. 배경 및 스타일
   - 전체 배경: 메인 서브 컬러 RGB(242,246,235)
   - 카드 및 입력 박스: 흰색 배경, 그림자 효과, 둥근 모서리
   - 안내 텍스트: RGB(158,158,158)
   - UI는 심플하고 신뢰성 있는 핀테크 분위기 유지

요청 스타일:
- 기존 마이페이지 레이아웃과 톤 앤 매너 일치
- 직관적이고 간단한 프로필 수정 경험 제공
- 친환경 핀테크 서비스 아이덴티티 반영
