1. 좌측 사이드 메뉴 
- 상단: 관리자 프로필 아이콘 
- 메뉴 항목: · 대시보드 설정 (→ 위젯 구성 커스텀 화면으로 이동) · 유저 관리 페이지 · 대중교통 연동 관리 · 커뮤니티 관리 · 통계 · 기부 관리 · 포인트샵 관리 · 제휴사 관리 · 설정 - 현재 활성화 메뉴: "대시보드 메인" - 배경: 포인트 컬러 RGB(255,206,85) - 활성화 메뉴 강조: 서브 컬러 RGB(86,130,101) 2. 본문 영역 (메인 보드판) - 화면 전체가 위젯 보드 레이아웃으로 구성 - 위젯 예시: · 통계 그래프 1 (일간 포인트 적립량) · 통계 그래프 2 (유저별 기여도 분포) · 기부 순위 (상위 기부자/단체 리스트) · 실시간 접속자 수 · 최근 게시물(커뮤니티 최신 글) · 문의글 현황 (처리 전/후 상태 구분) - 위젯은 카드 형태, 자유롭게 재배치 가능하다는 느낌 강조 3. 위젯 카드 디자인 - 흰색 배경, 그림자 효과, 둥근 모서리 - 제목 상단 고정 (예: “통계 그래프1”) - 내부: 그래프, 표, 리스트, 숫자 지표 표시 - 중요 수치 강조 색: 포인트 컬러 RGB(251,110,82) 4. 배경 및 스타일 - 전체 배경: 메인 서브 컬러 RGB(242,246,235) - 위젯 간 여백 충분히 확보, 격자형 정렬 - 텍스트: 기본 RGB(51,51,51), 보조 안내 RGB(158,158,158) - 그래프/차트 색상은 메인 & 서브 컬러 조합 요청 스타일: - 관리자 친화적인 대시보드 스타일 - 각 섹션을 위젯처럼 모듈화 → 커스터마이즈 가능성을 시각적으로 암시 - 데이터 가독성 강조 + 친환경 핀테크 서비스 아이덴티티 반영
